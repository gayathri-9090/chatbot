Welcome to the VXBot Tutorial

Explore the features of our chatbot and learn how to get the most out of it.

Getting Started

Follow these simple steps to use the chatbot:

  1. Type your message in the input field.
  2. Click the "Send" button to send your message to the chatbot.
  3. If you have a feedback type feedback: [your_feedback] and click send.

Chat Features

Our chatbot offers the following features:

  • Interactive chat interface with real-time responses.
  • Dark mode and light mode for your comfort.
  • Export chat conversations for future reference.
  • Mobile-friendly design for on-the-go usage.

Authorization

We value your input and expertise. Here's how you can help:

  1. You can provide answer only if you are authorized by developers.
  2. If you have an answer to a user's question, provide it using the "Provide Answer" button.
  3. When providing an answer, be sure to use '/' to separate multiple points if needed.
  4. If you have a file with the answer, provide an openly accessible Google Drive File ID or LMS link.

Theme and Navigation

Customize your experience with the following options:

  • Switch between dark and light themes using the theme toggle button.
  • Access the menu for quick navigation.
  • You can access the "History" link in the side menu only when you login.

Contact Form

Reach out to us using the contact form:

  • Click the contact icon to activate the form.
  • Provide your name, email, and message in the respective fields.
  • Submit the form for further assistance.
  • Confirmation or error messages will be displayed below the form.

Logout

If you wish to log out, click the Logout icon in the menu. You will be prompted to confirm your decision.

Upon confirmation, you will be redirected to the home page.
